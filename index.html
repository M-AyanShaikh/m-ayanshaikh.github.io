  • NBA Player Stats Explorer

    Python, Base64, Pandas, Streamlit

    This app performs extensive webscraping of NBA player stats data from 1950 to 2021. This app utilizes Python libraries such as Matplotlib, Base64, Pandas and Streamlit. The webscraping is done on popular basketball analytics website "Basketball Reference".

  • 'Haymath' - NBA News Twitter Bot

    Python, Tweepy

    This Twitter bot utilizes the Twitter API (through the use of the Tweepy library), to scan Twitter for tweets with keywords and retweets them, acting as a one-stop tweet shop for particular NBA players. This bot specifically retweets tweets related to upcoming NBA superstars LaMelo Ball and Zion Williamson. However, what sets this bot apart from others is that the bot can be redesigned to retweet any tweets with any kind of keyword with minimal changes!

  • Tuvver - Discord Bot

    Python

    Tuvver is a bot that utilizes the Discord API and a bot which users can add to their servers to assist with managing, updating and supervising a server. Tuvver can also be used by normal users in servers, and help with tasks such as playing music via YouTube, pinging friends and much more!
